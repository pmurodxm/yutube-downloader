# YouTube Downloader

Tkinter + yt-dlp asosidagi oddiy YouTube video/audio yuklovchi dastur (Windows uchun .exe).

## Xususiyatlar
- Video sifatini tanlash (360p – 1080p yoki eng yuqori)
- Faqat audio (mp3)
- Thumbnail ko'rsatish
- Progress bar + log
- Avtomatik yangilanish (GitHub Releases orqali)
- Qora-qizil-oq dizayn
- Telegram yangiliklari tugmasi

## Yuklab olish
Eng yangi versiyani [Releases](https://github.com/pmurodxm/yutube-downloader/releases) bo'limidan oling.

## Yangilanish jarayoni
1. Dastur ochilganda avtomatik yangi versiyani tekshiradi.
2. Yangi versiya bo'lsa "Yangilanish mavjud" oynasi chiqadi.
3. "Ha" desangiz → yangi .exe avtomatik yuklanib, eski fayl o'rnini bosadi.
4. Yangi versiyalar haqida: [Telegram kanal](https://t.me/CodeDrop_py)
